Installation

Get started in 4 simple steps

1

Download Installer

Get Wonderlands-Installer.jar from the button above

2

Run the JAR

Double-click to launch (requires Java 17+)

3

Select Minecraft Folder

Choose your .minecraft directory (auto-detected)

4

Launch & Play

Select "Wonderlands" profile in Minecraft Launcher
